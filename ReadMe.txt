# Event Log Pro - Event Management System

A comprehensive event management system for schools to track field trips, tours, and student events.

## Features
- Event tracking with unique IDs (YYYY-XXX format)
- Student roster management
- Medical information tracking
- Permission slip monitoring
- Automated report generation
- Dark mode support
- Google Sheets integration

## Setup
1. Deploy the Google Apps Script
2. Configure the Web App URL in Settings
3. Add your Google Sheet ID
4. Start managing events!
